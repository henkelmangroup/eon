Download¶

EON is released under the BSD License.

The EIGEN library, which is included with EON, is released under the Mozilla Public License.

The latest version of the code can be obtained from GitHub via HTTPS with the following command:

git clone https://github.com/henkelmangroup/eon.git

If you prefer to use SSH, then the command becomes:

git clone git@github.com:henkelmangroup/eon.git

This will fetch a copy of the latest code to a local directory named eon. For more information on GitHub read its online documentation.

The eon code can also be download directly at tarfile..
