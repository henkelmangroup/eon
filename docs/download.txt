
.. _download:

================================
Download
================================

EON is released under the `BSD License`_.

The EIGEN library, which is included with EON, is released under the Mozilla Public License.

The latest version of the code can be obtained from GitHub via HTTPS with the following command:

.. code-block:: sh

    git clone https://github.com/henkelmangroup/eon.git

If you prefer to use SSH, then the command becomes:

.. code-block:: sh

    git clone git@github.com:henkelmangroup/eon.git

This will fetch a copy of the latest code to a local directory named eon. 
For more information on GitHub read its `online documentation`_.

The eon code can also be download directly at `tarfile`_.

.. _online documentation: https://docs.github.com/en/get-started
.. _BSD License: https://opensource.org/licenses/BSD-3-Clause
.. _tarfile: https://github.com/henkelmangroup/eon/releases/latest/download/eon.tar.gz
